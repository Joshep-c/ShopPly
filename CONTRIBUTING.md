# Contributing to ShopPly

Repo: https://github.com/Joshep-c/ShopPly

Gracias por tu interés en contribuir! Sigue estas pautas para facilitar revisiones rápidas:

1. Fork & Branch

- Crea una rama con un prefijo: `feat/`, `fix/`, `docs/`.

2. Código

- Mantén estilo Kotlin idiomático.
- Escribe tests para cambios de lógica.

3. Commits

- Mensajes cortos y descriptivos. Ej: `fix(auth): validate email in register`.

4. PR

- Describe el cambio y cómo probarlo.
- Añade capturas o pasos para reproducir si aplica.

Nota: abre Pull Requests contra la rama `main` (o la rama de trabajo indicada en la issue). Incluye en la PR:

- Resumen de cambios
- Cómo probar localmente
- Capturas o logs si aplica

Si quieres, puedo añadir una plantilla de PR (`.github/PULL_REQUEST_TEMPLATE.md`) para estandarizar el proceso.

5. Revisiones

- Responde a comentarios y actualiza la PR.

6. Seguridad

- No subir secretos o credenciales al repo.
