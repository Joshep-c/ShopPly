# domain module

Propósito

- Contiene las entidades del negocio, casos de uso y reglas de negocio puras.

Qué incluir

- Entidades inmutables y DTOs.
- Casos de uso (use-cases) que orquesten operaciones que no dependen de Android.

Beneficios

- Facilita pruebas unitarias y desacoplamiento de la capa de presentación y datos.
